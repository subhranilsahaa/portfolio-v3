# Portfolio v3 - Production Ready with Supabase

A premium, performant, and deeply customizable portfolio built for modern full-stack developers.

## Setup & Local Development
1. Replace `YOUR_PUBLIC_ANON_KEY_HERE` in the `.env` file with your actual Supabase Anon key.
2. Run `npm install`
3. Run `npm run dev`

## Admin Panel
Navigate to `/admin` or click login.
**Default Password:** `portfolio2025`
